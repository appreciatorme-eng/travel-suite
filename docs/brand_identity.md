## 2. Typography

### Headings: `Cormorant Garamond`
*   **Style**: Serif, Elegant, "Adventure Magazine" feel.
*   **Usage**: Hero titles, Itinerary names, "Explore Beyond Comfort Zones".
*   **Google Fonts**: [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond)

### UI / Body: `Poppins`
*   **Style**: Sans-serif, Clean, Modern.
*   **Usage**: Buttons, Navigation, Input fields, Body text.
*   **Google Fonts**: [Poppins](https://fonts.google.com/specimen/Poppins)
